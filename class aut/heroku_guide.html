Method 2: Command Line

For advanced users

First install Heroku CLI: Download Here
Commands to run:
# Login to Heroku
heroku login

# Create app
heroku create smart-physics-hub-yourname

# Initialize Git
git init
git add .
git commit -m "Deploy Smart Physics Hub"

# Deploy
git push heroku main

# Open app
heroku open
Or run our automated script: deploy_heroku.bat
